# Amazon Consumer Insights & Predictive Model 📊🚀

## 📝 Project Overview
In this project, I developed an end-to-end Data Science pipeline to analyze consumer behavior and product ratings on Amazon. By leveraging advanced statistical methods and machine learning algorithms, I uncovered key insights into user reviews and built a model to accurately predict customer satisfaction and rating trends.

---

## 🎯 My Objectives
* **Data Engineering & Preprocessing:** I cleaned and structured textual and numerical customer review data to prepare it for advanced analysis.
* **Statistical Inference:** I applied robust statistical hypothesis testing to determine the core factors that influence review helpfulness.
* **Predictive Modeling:** I built and optimized a high-performance Machine Learning model to classify and forecast consumer satisfaction.

---

## 📊 My Methodology

### 1. Data Preprocessing & Feature Engineering
* I handled missing values and conducted rigorous data cleaning to ensure data quality.
* I performed feature engineering, including computing exact review length and depth metrics using optimized `lambda` functions.

### 2. Statistical Analysis & Hypothesis Testing
* I computed key descriptive statistics (Mean, Median, and Mode) to map the distribution of product scores.
* I executed **Hypothesis Testing (T-Test)** to identify statistically significant differences across product groups and quantify features driving review helpfulness.

<h3>3. Machine Learning & Predictive Analytics</h3>
* I developed and trained a classification model utilizing the **Random Forest Classifier** algorithm.
* I conducted a thorough model evaluation to ensure robust predictive precision on unseen consumer datasets.

---

* **Programming Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Statistical Analysis:** SciPy (Stats module)
* **Data Visualization:** Matplotlib, Seaborn
